# Student Usage Guide — GSA Gateway

Welcome! GSA Gateway is your AI-assisted companion for graduate student life at NJIT. Here's everything you need to know.

## Getting Started

1. **Join the GSA Discord server** — ask a GSA officer for the invite link, or find it on Highlander Hub.
2. **Go to any GSA channel** (e.g. `#gsa-general` or `#bot-commands`).
3. **Type `/help`** to see all available commands.

---

## Commands

### `/ask [question]`
Ask anything about graduate student life at NJIT.

**Examples:**
- `/ask How do I apply for OPT?`
- `/ask What funding is available for research?`
- `/ask How do I file a grievance?`

The bot searches its knowledge base using fuzzy matching. If it can't find a good answer, it will suggest contacting a GSA officer.

### `/events`
See all upcoming GSA events — workshops, coffee hours, networking nights, and more.

### `/event [name]`
Get full details for a specific event.
- Example: `/event Research Mixer`

Partial names work: `/event research` will find "Graduate Research Mixer".

### `/initiative`
Submit an idea, proposal, or request to GSA.
- A private form opens — fill in your title, description, and category.
- **Anonymous by default** — your Discord ID is never stored in plain text.
- Optionally include contact info if you want GSA to follow up with you.

**Great for suggesting:**
- New workshops or events
- Policy changes you'd like to see
- Resources that should be added
- Campus issues affecting graduate students

### `/feedback [message]`
Send a private anonymous message to GSA officers.

**Example:** `/feedback The library closes too early on weekends for graduate students.`

Your message is visible only to GSA officers. Your identity is never linked to it.

### `/resources [category]`
Browse NJIT resources organized by topic.

- `/resources` — see all 8 categories
- `/resources academic` — academic support links
- `/resources funding` — scholarships and fellowships
- `/resources wellness` — counseling and health resources
- `/resources international` — visa, OPT, tax help

### `/contact [role]`
Look up GSA officers and NJIT offices.

- `/contact` — see the full directory
- `/contact VP Academic Affairs` — specific officer info
- `/contact Counseling Center` — campus office info

### `/help`
Show a summary of all commands (private — only you can see it).

---

## Privacy

- Your Discord user ID is **hashed** before storage — it cannot be read or reversed by GSA officers.
- Question text and feedback messages are stored in plain text for officer review, but with **no link to your identity**.
- When you submit an initiative, your contact info is only stored if you explicitly provide it and answer "yes" to the contact question.

---

## Tips

- Commands work via autocomplete — start typing `/` and Discord will show suggestions.
- Partial names work for events and contacts — no need to type the full name.
- If the bot doesn't answer your question, try rephrasing it or use `/contact` to reach an officer directly.
- The bot is rate-limited to 5 commands per minute to keep the server responsive.
