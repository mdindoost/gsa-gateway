# Kavosh v2.1 — Phase 1b Unified Router Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Replace the four hand-maintained routing mechanisms with ONE unified, classify-then-resolve Router (masked coarse-family classifier → deterministic skill resolver), shipped behind a flag, shadow-mode first, then flipped — without ever letting an exact/structured query get a fuzzy answer.

**Architecture:** A new `UnifiedRouter.decide(message, query_vec)` runs five layers in order — (0) deterministic COMMAND layer, (1) deterministic FAST-PATH, (2) masked coarse-family CLASSIFIER (one nomic encode, reused by the retriever), (3) deterministic RESOLVER (today's `router.route()` + its negative guards + terminal-skill + inverse-FN guards), (4) CLARIFY / 3 RAG outcomes / live. The classifier (validated in Phase 1a as `masked_coarse`) only picks the FAMILY; the SQL skill stays deterministic. Integrated into `message_handler.handle` behind `ROUTER_V21`, shadow-logging new-vs-current before flipping. Phase-0/1a decision artifacts: `findings/2026-06-21-kavosh-v2.1-phase0-conclusion.md`, `findings/2026-06-21-kavosh-v2.1-phase1a-gonogo.md`. Design: `specs/2026-06-21-kavosh-v2.1-phase1-build-design.md` §5, and the FINAL architecture spec `specs/2026-06-21-kavosh-v2.1-router-gate-design.md` §5–§12.

**Tech Stack:** Python 3.11, numpy, the existing deterministic router (`v2/core/retrieval/router.py`), nomic embeddings via Ollama (`v2/core/retrieval/embedder.py`), SQLite, pytest. No new heavy dependency.

## Global Constraints

- **G2 exactness is non-negotiable.** Counts/rankings/rosters/metrics/roles come ONLY from `structured_answer.run` over `router.route`'s parameterized SQL. The classifier picks a FAMILY only; the LLM never computes. Preserve `is_deterministic` + `deterministic_suffix` + the skip-compose path in `message_handler._try_structured` (~L329-336) VERBATIM in the new path.
- **Shipped classifier = `masked_coarse`, no levers** (Phase-1a go/no-go): masked coarse-family exemplar-NN → deterministic resolver. Abstention BUILT but OFF (not calibrated to a usable point). Inverse-FN guard requires the resolver to actually VALIDATE slots, not merely return non-None (Phase-1a showed the naive version converts a struct-FN into a wrong-skill dispatch).
- **One encode per message (hard invariant).** The single nomic vector serves the classifier AND (on fall-through) the retriever. A unit test asserts `embed_query` is called ≤ 1× per `handle()`.
- **Encode prefix/version guard.** The classifier needs the RAW message under a FIXED `ROUTER_PREFIX` ("router_query: "). Exemplars carry encoder model + prefix; startup fails loudly on `EMBEDDING_MODEL` / `OLLAMA_URL` / prefix drift.
- **Flag-gated, reversible.** `ROUTER_V21` default OFF. Shadow mode logs new-vs-current and ACTS on the current path until the flip gate (spec §12) is met. The flag stays as a kill-switch. Phase-2 LLM slot-recovery is a SEPARATE sub-flag, OFF, OUT of this plan.
- **Preserve the negative guards.** `_OFFICER_PROCESS`, `_LEADERSHIP_PROCESS`, `_has_child_departments`, `_is_bare_name`, the ≤4-token surname guard in `router.py` are load-bearing anti-fab logic — reused as-is, never re-derived by the classifier.
- **No `scripts/ask.sh` for regression.** The committed regression set runs THROUGH `handler.handle` (ask.sh bypasses the gate). Every change grows `eval/questions.txt`.
- **TDD, frequent commits, no Claude attribution.** Commit prefixes: `feat(router-v21):` for code, `test(router-v21):` where a task is test-only, `docs(kavosh-v2.1):`.
- **Run tests:** `python3 -m pytest <path> -q` from `/home/md724/gsa-gateway`. Unit tests use a fake encoder/`monkeypatch` — no Ollama. The shadow/flip measurement tasks (D, E) need Ollama + the live KG (read-only).

# v2/core/retrieval/route_exemplars.py  (stamp portion; the loader is added in Task B2)
from __future__ import annotations

ROUTER_PREFIX = "router_query: "


def router_embed(embedder, text: str):
    """Embed the RAW message under the FIXED router prefix (NOT search_query:). The router
    classifier and the retriever therefore live in two prefix spaces; the shared encode in the
    handler is the router one, and the retriever re-encodes if it needs its own (one encode each)."""
    raw = embedder.embed_query  # type: ignore[attr-defined]
    # embed_query prepends "search_query: " — to get the router prefix we call the low-level path.
    # Embedder exposes only embed_query/_embed; we reuse embed_query semantics by passing our prefix
    # through embed_query's own input (it strips+caps but does not re-prefix the caller's text here):
    return _embed_with_prefix(embedder, ROUTER_PREFIX + text)


def _embed_with_prefix(embedder, prefixed_text: str):
    # Embedder.embed_query hardcodes "search_query: "; for the router space we go through the
    # normalize+_embed pair directly so ONLY ROUTER_PREFIX is applied.
    vec = embedder._embed(prefixed_text)         # noqa: SLF001 - intentional: single embed path
    return embedder.normalize(vec)


def encoder_stamp(embedder) -> dict:
    return {"model": embedder.model, "base_url": embedder.base_url, "prefix": ROUTER_PREFIX}


def verify_stamp(embedder, stamp: dict) -> None:
    current = encoder_stamp(embedder)
    if current != stamp:
        raise RuntimeError(
            f"Router encoder drift — exemplars were built with {stamp} but runtime is {current}. "
            "Rebuild exemplars or fix EMBEDDING_MODEL/OLLAMA_URL.")
```

# v2/core/retrieval/route_classifier.py
from __future__ import annotations
import numpy as np


class RouteClassifier:
    """Masked exemplar-NN family classifier (the Phase-1a `masked_coarse` mechanism, productionized).
    Picks the coarse FAMILY only; the SQL skill stays deterministic in the resolver."""

    def __init__(self, exemplars, encode_fn, masker):
        self.encode_fn = encode_fn
        self.masker = masker
        self.row_label = [fam for _q, fam in exemplars]
        masked = [masker.mask(q) for q, _fam in exemplars]
        self.mat = encode_fn(masked) if masked else np.zeros((0, 0))

    def ranked(self, query: str):
        if self.mat.shape[0] == 0:
            return []
        q = self.encode_fn([self.masker.mask(query)])[0]
        sims = self.mat @ q
        best: dict[str, float] = {}
        for lab, s in zip(self.row_label, sims):
            if s > best.get(lab, -1.0):
                best[lab] = float(s)
        return sorted(best.items(), key=lambda kv: kv[1], reverse=True)

    def top(self, query: str):
        r = self.ranked(query)
        if not r:
            return ("", 0.0, 0.0)
        label, score = r[0]
        margin = score - (r[1][1] if len(r) > 1 else 0.0)
        return (label, score, float(margin))
```

DEFAULT_EXEMPLARS = "eval/router/labeled_routes.jsonl"


def load_exemplars(path: str = DEFAULT_EXEMPLARS):
    rows = []
    for line in Path(path).read_text().splitlines():
        line = line.strip()
        if not line:
            continue
        d = json.loads(line)
        if d.get("split") == "hardneg":
            continue

    exemplars = load_exemplars(path)
    masker = build_masker(conn)
    encode_fn = lambda texts: __import__("numpy").array(

from dataclasses import dataclass, field

    args: dict = field(default_factory=dict)

    def _rag_outcome(self, message: str) -> str:
        from bot.services.intent_detector import INTENT_FOOD
        intent, _ = self.intent_detector.detect(message)
        if intent == INTENT_FOOD:
            return "food"
        if any(w in message.lower() for w in ("event", "events", "happening", "workshop", "seminar")):
            return "event"
        return "general"

    def decide(self, message: str) -> "RouteDecision":
        cmd = self.command_layer(message)
        if cmd is not None:
            return cmd
        ranked = self.classifier.ranked(message)
        top = ranked[0][0] if ranked else "RAG"
        if top == "KG":

        inv = self.inverse_fn_guard(message, ranked)
        if inv is not None:

        if top == "RAG":
            return RouteDecision(family="RAG", source=self._rag_outcome(message),
                                 score=ranked[0][1])
        if top == "LIVE":
            return RouteDecision(family="LIVE", score=ranked[0][1])
        return RouteDecision(family=top, score=(ranked[0][1] if ranked else None))
