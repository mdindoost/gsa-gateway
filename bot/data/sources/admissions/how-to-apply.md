# How to Apply for Graduate Admission

Apply online through NJIT's graduate application — you create a user ID and password and can save and return to it later. Check your specific program's page for any additional requirements. You may apply to **only one degree program at a time**. Apply at connect.njit.edu/apply.

## Application checklist — Master's
- Application for admission and the **$75 non-refundable application fee**
- Transcripts from all colleges and universities attended
- GRE/GMAT scores (optional for most programs — see the Test Scores overview)
- TOEFL / IELTS / Duolingo / PTE scores (international applicants without a U.S. degree)
- Letter of recommendation (optional for most master's; required for Architecture and Urban Design)
- Portfolio (Architecture and Design applicants only)

## Application checklist — Ph.D.
- Application and the $75 fee
- Transcripts and proof of degree completion from all institutions attended
- GRE/GMAT scores (required for some doctoral programs)
- TOEFL / IELTS / Duolingo / PTE scores (international applicants without a U.S. degree)
- **Three letters of recommendation**
- **Statement of purpose**

## Application checklist — Graduate Certificate
- Application and the $75 fee, official transcripts, a **resume**, GRE/GMAT (optional for select programs), and an English test score if requested.

Unofficial copies of documents are accepted to complete your application; admitted students must submit official documents before enrolling. Questions: Office of University Admissions, admissions@njit.edu.
