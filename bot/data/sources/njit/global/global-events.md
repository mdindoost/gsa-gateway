The Office of Global Initiatives hosts, co-hosts, and participates in many events, workshops, and webinars throughout the year.  The list of events is updated regularly so keep checking for the most updated list of events!

(For the current figure, see the live page: https://www.njit.edu/global/events)

(For the current figure, see the live page: https://www.njit.edu/global/events)
