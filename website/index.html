GSA Gateway
Your voice, amplified.

What is GSA Gateway?

One bot. One website. Everything a graduate student needs.

Knowledge Base

Ask Anything

Use /ask in Discord to search our curated FAQ — academic policies, funding, wellness, and more. Powered by fuzzy-search with optional AI enhancement.

Events

Stay Connected

Browse upcoming GSA events with /events. Workshops, mixers, networking nights, and coffee chats — all in one place.

Initiatives

Drive Change

Have an idea? Use /initiative to submit it directly to GSA officers through an anonymous Discord form. Your voice matters.

Resources

Find Support

Quickly access curated resources in 8 categories: academic, funding, wellness, international, research, housing, transportation, and campus life.

Privacy

Anonymous by Default

Your Discord ID is never stored in plain text. All identifiers are SHA-256 hashed. Contact info is only stored when you explicitly opt in.

Open Source

Transparent & Free

GSA Gateway is open source, built with Python and discord.py. No paid APIs. No subscriptions. Runs entirely on free infrastructure.

Discord Commands

All commands are slash commands — type / in any GSA Discord channel.

CommandDescription
/ask [question]Search the GSA knowledge base for answers
/eventsList all upcoming GSA events
/event [name]Get full details for a specific event
/initiativeSubmit an idea or proposal to GSA (anonymous)
/feedback [message]Send private anonymous feedback to officers
/resources [category]Browse NJIT resources by category
/contact [role]Find contact info for officers and offices
/helpShow all available commands
